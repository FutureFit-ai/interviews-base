# Node Coding Interview Starter

Welcome! Thanks for taking the time to interview with FutureFit. This repo is a starter node project
we'll be using over the course of the interview. Prior to the interview ensure you can clone this
repo and run the tests (see instructions below) to ensure your environment is setup properly.

This repo is just a starting point, when the interview starts your interviewer will share the actual
problem with you.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: LTS version (20.x or higher recommended) ([Download here](https://nodejs.org/))
- **npm**: Version 10.x or higher (comes with Node.js)

You can verify your installations by running:

```bash
node --version
npm --version
```

## Setup

The project includes:

- **TypeScript**: For type-safe JavaScript development
- **Jest**: For testing framework
- **ts-jest**: For running TypeScript tests with Jest

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run tests:

```bash
npm test
```
